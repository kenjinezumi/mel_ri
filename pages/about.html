About:


Purpose:

This tool is aimed at providing a quick and easy solution to transform Excel scorecard into consumable CSV files.

Scope:

The tool is built with several constraints:

  • It should be OS agnostic (work on Mac as well as Windows)
  • It should not be package dependent (the end-user should not have to download any library)
  • It should be easy to use and not hosted on a server

    • Tech stack

    The tech stack consists of a client JavaScript web page using CDN libraries such as ExcelJS.

    Output

    The script ingests multiple Excel files and transform them into Text files in a CSV format.

    Each tab of a CSV file will output one file. For example, if you load 60 Excel files with 5 similar tabs, it will output 5 files corresponding to your tabs with the merged files. E.g. 60 excel files with tabs "safety", "people centered" will deliver 2 text files with the merged data.

    How to use it

    1. Download the tool on your local machine.
    2. Go into the folder "MEL_RI" and open "index.html" in your browser - ideally Google Chrome
    3. Click on "choose files" and select your Excel files.
    4. Clik on "allow multiple downloads"
    5. The file will then be downloaded in your browser.

    IMPORTANT: constraints

    Due to the nature of the Excel files - several constraints in terms of format must be respected

    1. Name of the excel file: the excel files must follow the strict format "2022 Q4 Health scorecard_COUNTRYNAME_PROJECTCODE.xlsx"
    2. Structure of the excel file: the structure of the excel file must follow the templated one.
